# Requisitos Funcionales – Smart Nurse IA

RF-01  
El sistema debe permitir registrar auxiliares y enfermeros con nombre, documento y área.

RF-02  
El sistema debe permitir asignar turnos mensuales a los profesionales de salud.

RF-03  
El sistema debe validar que no existan conflictos de horarios en los turnos.

RF-04  
El sistema debe permitir solicitar intercambio de turnos entre dos profesionales.

RF-05  
El sistema debe registrar la firma de ambos profesionales para validar el intercambio.

RF-06  
El sistema debe enviar la solicitud al jefe de enfermería para aprobación.

RF-07  
El jefe de enfermería debe poder aprobar o rechazar solicitudes.

RF-08  
El sistema debe validar que no se excedan las horas laborales mensuales.

RF-09  
El sistema debe registrar el historial de cambios de turnos.

RF-10  
El sistema debe permitir consultar turnos mensuales.
