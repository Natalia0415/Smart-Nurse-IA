# Tecnologías Utilizadas

## Frontend

HTML
CSS
JavaScript

Permiten construir la interfaz de usuario del sistema.

---

## Backend

Lenguajes de programación utilizados para la lógica del sistema.

Permiten procesar información como:

* validación de turnos
* cálculo de horas laborales
* gestión de solicitudes

---

## Base de datos

Se utiliza una base de datos relacional para almacenar:

* personal de enfermería
* turnos
* solicitudes de intercambio
* registros de cambios

---

## Control de versiones

Git y GitHub se utilizan para gestionar el código del proyecto.

Permiten mantener historial de cambios y trabajo colaborativo.

---

## Justificación

Estas tecnologías permiten construir un sistema escalable, seguro y fácil de mantener.
