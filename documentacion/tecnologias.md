# Tecnologías Utilizadas en el Desarrollo del Sistema

En el desarrollo del sistema **Smart-Nurse IA** se utilizaron diferentes tecnologías orientadas a la construcción de una aplicación web capaz de gestionar turnos de enfermería, solicitudes de intercambio y validaciones de horas laborales.

A continuación se describen las principales tecnologías utilizadas y la justificación de su selección.

---

## Frontend

Tecnologías utilizadas:

- HTML
- CSS
- JavaScript

Descripción:

Estas tecnologías se emplean para el desarrollo de la interfaz de usuario del sistema. Permiten construir las páginas web mediante las cuales los profesionales de enfermería y los jefes de área interactúan con la aplicación.

HTML se utiliza para estructurar el contenido de las páginas, CSS para el diseño visual y la organización de los elementos, y JavaScript para implementar funcionalidades dinámicas como validaciones de formularios, visualización de turnos y gestión de solicitudes.

Justificación:

Estas tecnologías fueron seleccionadas debido a su amplia adopción en el desarrollo web, su compatibilidad con todos los navegadores modernos y su capacidad para crear interfaces interactivas y accesibles para los usuarios del sistema.

---

## Backend

Tecnologías utilizadas:

Lenguajes de programación y herramientas orientadas a la lógica del sistema.

Descripción:

El backend se encarga de procesar la información del sistema y ejecutar las reglas de negocio. Entre sus funciones principales se encuentran:

- Validación de turnos asignados
- Cálculo de horas laborales mensuales
- Gestión de solicitudes de intercambio de turnos
- Control de permisos según el rol del usuario
- Comunicación con la base de datos

Justificación:

La implementación de un backend permite separar la lógica del sistema de la interfaz de usuario, facilitando el mantenimiento del software, la escalabilidad del sistema y la implementación de controles de seguridad y validación de datos.

---

## Base de Datos

Tecnología utilizada:

Base de datos relacional.

Descripción:

La base de datos permite almacenar y gestionar la información necesaria para el funcionamiento del sistema, incluyendo:

- Información del personal de enfermería
- Identificación del profesional (documento)
- Tipo de profesional (auxiliar o enfermero)
- Registro de turnos mensuales
- Solicitudes de intercambio de turnos
- Historial de cambios y aprobaciones

Justificación:

Se seleccionó una base de datos relacional debido a su capacidad para gestionar relaciones entre diferentes entidades del sistema, garantizando integridad de los datos, consistencia de la información y facilidad para realizar consultas estructuradas.

---

## Control de Versiones

Tecnologías utilizadas:

- Git
- GitHub

Descripción:

Git se utiliza como sistema de control de versiones para registrar los cambios realizados en el proyecto durante su desarrollo. GitHub funciona como repositorio remoto donde se almacena el código y la documentación del sistema.

Justificación:

El uso de estas herramientas permite mantener un historial de cambios del proyecto, facilitar el trabajo colaborativo entre los integrantes del equipo y asegurar la correcta gestión del ciclo de vida del software.

---

## Conclusión

Las tecnologías seleccionadas permiten construir un sistema web estructurado, escalable y fácil de mantener. Además, facilitan la implementación de buenas prácticas de desarrollo de software, incluyendo control de versiones, organización modular del sistema y gestión adecuada de la información.
