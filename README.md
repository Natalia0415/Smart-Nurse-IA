# Smart-Nurse IA

Sistema inteligente para la gestión y optimización de turnos del personal de enfermería.

---

## Descripción del proyecto

Smart-Nurse IA es un sistema diseñado para facilitar la planificación y administración de turnos del personal de enfermería dentro de instituciones de salud.

El sistema permite registrar auxiliares y enfermeros, gestionar calendarios de turnos, controlar las horas laborales mensuales y administrar solicitudes de intercambio de turnos entre profesionales, garantizando que dichos cambios sean aprobados por el jefe de enfermería del área correspondiente.

El objetivo principal es mejorar la organización del personal, reducir errores humanos en la asignación de turnos y asegurar el cumplimiento de las políticas laborales del hospital.

---

## Problema identificado

En muchos centros hospitalarios la asignación de turnos se realiza de forma manual o mediante hojas de cálculo, lo que genera problemas como:

* errores en la planificación de turnos
* dificultad para controlar las horas trabajadas
* intercambios informales de turnos
* falta de trazabilidad en las aprobaciones

Smart-Nurse IA busca automatizar este proceso y mejorar la gestión del personal.

---

## Funcionalidades del sistema

El sistema permite:

* registrar auxiliares y enfermeros
* almacenar documento de identificación del personal
* controlar horas laborales mensuales
* asignar turnos mediante códigos
* visualizar calendario mensual de turnos
* solicitar intercambios de turnos
* confirmar intercambios entre profesionales
* aprobar intercambios por jefe de enfermería

---

## Roles del sistema

El sistema maneja diferentes roles:

Auxiliar de enfermería
Puede consultar turnos y solicitar intercambios.

Enfermero
Puede gestionar turnos y participar en intercambios.

Jefe de enfermería
Tiene permisos para aprobar o rechazar intercambios.
